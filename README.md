# Student-Performance-Prediction
📌 Overview

This project predicts whether a student will pass or fail based on academic and personal factors using Machine Learning techniques. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation.

# Features
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Correlation Analysis
Random Forest Classification Model
Model Performance Evaluation
Data Visualization

# Project Workflow
Data Collection

Model Evaluation
Results Visualization
# Evaluation Metrics
Accuracy Score
Precision
Recall

# Results

The Random Forest model successfully predicted student pass/fail outcomes by identifying the most influential academic and behavioral factors affecting performance.
